# CNCF (cncf)
The Cloud Native Computing Foundation (CNCF) is part of the Linux Foundation and hosts critical components of the global technology infrastructure, including Kubernetes, Prometheus, and Envoy. CNCF fosters collaborative development and adoption of cloud native technologies.
